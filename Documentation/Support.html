How does this work?
  • Select a folder containing either files, or subfolders.
  • Type in a new name, or select a new name from the Templates.
  • Hit Process, to rename!
  • Select Undo, to undo a rename.
Can I save Templates?
Yes! You can export & import templates. You can even create templates from existing files in the view!
